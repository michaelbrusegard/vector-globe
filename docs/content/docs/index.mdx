## Initialization

To initialize the Vector Globe, import the `VectorGlobe` class and create a new instance.

```javascript
import VectorGlobe from 'vector-globe';

const globe = new VectorGlobe();
```

You can pass an optional container element to the constructor to render the globe in it:

```javascript
const container = document.querySelector('div');

const globe = new VectorGlobe(container);
```

If not provided, the globe will be rendered in the body element.

## Start & Stop

To start the globe, call the `start` method on the instance.

```javascript
globe.start();
```

And to stop the globe, call the `stop` method.

```javascript
globe.stop();
```

<Callout type='warn'>
  Call `stop` before removing the globe from the DOM to avoid memory leaks and clean up resources.
</Callout>
